# Banana King
Banana King Nerver Absent! Timed execution.

## Installation
python3 version

```
$ pip install banana-king
```

optional arguments:
  -h, --help            show this help message and exit
  -t TIME, --time TIME  set time to execute, format: "H:M:S"
  -f, --foreground      execute this program foreground
  -b, --background      execute this program background
  -p PROGRAM, --program PROGRAM
                        program path

Example:
  1. banana-king -p "ls /"
  2. banana-king -p "ls /" -t 22:23
  3. banana-king -p "ls /" -t 22:23 -b
